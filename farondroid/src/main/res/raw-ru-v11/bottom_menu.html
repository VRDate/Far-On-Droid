Нижняя панель содержит несколько полезных операций.

Элементы, которые не помещаются на экране, доступны в выпадающем меню (с помощью клика по элементу Еще).
Удерживая Alt вы включаете режим мульти выбора, т.е. клик по файлу/директории добавит его/ее в список выбранных файлов.
Опция Выбрать используется для выбора файлов с помощью паттерна.
Опция Меню открывает контекстное меню со списком доступных операций.
Опция Сеть открывает контекстное меню со списком доступных сетевых хранилищ.
Опция Приложения заменяет активную панель панелью с установленными приложениями
Опция Новый используется для создания файла или директории.
Попощь - открыть экран помощи.
Настройки - открыть экран настроек.
Опция Детали заменяет неактивную панель панелью с информацией об элементах активной панели
Опция Сравнение используется чтобы показать отличие между директориями. После завершения сравнения будут выбраны отличающиеся файлы.
Опция Поиск используется для продвинутого поиска.
Опция Закладки используется, чтобы показать диалог со списком закладок (это список также доступен в директории Bookmarks в корне файловой системы).
Опция Приложения заменяет активную панель панелью со списком установленных приложений.
