File viewer is a tool for viewing and editing text files. Default mode - view, i.e. text file is opened for viewing. Edit operation need to be enabled manually. Replace operation is available for both modes.
Goto option is used to go to defined line number (or percent). Find option is to find selected text in file. Found text will be selected by yellow background. A few search params are supported.
Replace actually doing the same as Find option but also replaces found text.
Edit/View button is used to switch in viewer modes.
View mode is default and used for text reading. In Edit mode document became editable. Save operation is performed in background and takes some time.
Encoding button is used to open Select encoding dialog where you can select encoding (and save it as default) for current file. Default value is "UTF-8".

For very large files only View mode is available (with some limitations). File will be broken on a few fragments. Click on ... lines at the end or begin of current fragment to navigate to next or previous fragment appropriately.
