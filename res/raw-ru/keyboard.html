Far On Droid имеет базовую поддержку внешних клавиатур.

Удерживайте Shift и нажимайте "Вверх"/"Вниз" для выделения файлов.
Используйте Tab для переключения между панелями.
F2 - открыть диалог с доступными действиями.
F5 - открыть диалог копирования.
F6 - открыть диалог перемещения/переименования.
F7 - открыть диалог создания файла/директории.
F8 - открыть диалог удаления.
F9 - открыть диалог поиска.
F10 - открыть диалог помощи.
